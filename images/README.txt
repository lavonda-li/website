Image slot manifest
===================

Drop real photos into this folder using the filenames below. Once a file is
present, uncomment the matching <img> tag in the HTML (or replace the
placeholder <span> label) and the tile will render the image.

Notes
-----
- Square (1:1) crops work best for the Home collage and Work 2x2.
- Portrait (4:5) crops work for home-hero-grad.
- Landscape (~4:3) for home-hero-portrait.
- JPG or WebP both fine; keep each under ~400KB if possible.
